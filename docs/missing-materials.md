# Missing Materials Checklist

Это не список общих пожеланий, а конкретные реальные материалы, которых сейчас не хватает для финальной публикации.

## 1. Hero и публичные секции

- Реальное hero-фото фасада вместо service-fallback `hero-facade-scene.svg`.
- Реальное фото или коллаж для блока "Описание продукта" вместо `product-feature-board.svg`.
- Реальные фото для карточек галереи:
  - зернистая фактура;
  - гладкая фактура;
  - клинкерная фактура;
  - комбинации цвета панели и шва;
  - светлые фасадные решения;
  - контрастные фасадные решения.

## 2. Каталог и карточки товаров

Для каждого товара всё ещё нужны реальные материалы по 4 обязательным сценариям:

- фото панели издалека;
- крупный план фактуры и шва;
- фото сбоку с видимой толщиной;
- фото дома с этой расцветкой.

Сейчас эти сценарии закрыты service-fallback файлами:

- `product-panel-far.svg`
- `product-panel-closeup.svg`
- `product-panel-side.svg`
- `product-house-example.svg`

## 3. Объекты / дома

- Реальные фото готовых объектов для витрины примеров домов.
- Сейчас используются service-fallback визуалы:
  - `gallery-light-facade.svg`
  - `gallery-contrast-facade.svg`

## 4. Самостоятельный монтаж

- Реальное видео или хотя бы обложка видео процесса монтажа.
- Сейчас используется service-fallback `self-install-scene.svg`.

## 5. Контакты и соцсети

- Реальная ссылка на Telegram.
- Реальная ссылка на VK, если канал действительно используется.

Сейчас публичная часть скрывает служебные ссылки-заглушки, а в админке такие поля помечаются warning-индикаторами.
